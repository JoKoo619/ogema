Directory for OGEMA 2.0 related code/data that does not strictly belong to OGEMA.

content/subdirectories:

eclipse-formatter:
    Setup for the code formatting. Used by the Maven formatting tool as 
    configured in OGEMA's main pom.xml

equinox-console-scripting:
    A tool for running scripts on the Equinox Console. Contains sources and
    a Maven build setup.

license:
    License header to be used by the licensing tool. The use of this header
    is configured in OGEMA's main pom.xml.
